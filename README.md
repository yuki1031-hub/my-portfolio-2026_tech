# My Portfolio 2026 🚀

## はじめに
私のポートフォリオサイトのソースコードです。
自己紹介、経歴、スキル等に加えて、実際に制作したアプリケーション（Deployed Projects）を収録しています。

## 収録プロジェクト (Deployed Projects)

### 1. Figma to YAML (Figma Plugin)
* **概要**: 選択したFigmaのフレームやグループのレイヤー階層を読み取り、構造化された「YAML形式」のデータに自動変換（シリアライズ）するプラグインです。デザインとエンジニアリングのギャップを埋め、コンポーネント生成などの開発パイプラインを大幅に効率化します。
* **技術**: Figma Plugin API, JavaScript / TypeScript

### 2. FoodSwipe（今日、何食べる？）
* **概要**: 日常の「今日のご飯が決まらない」という悩みを解決するため、直感的なスワイプ操作（LIKE / NOPE）で食事をサクッと決定できるWebアプリケーションです。
* **技術**: Next.js (App Router), React, Tailwind CSS

### 3. 性格診断チャットボット (Personality Diagnosis Chatbot)
* **概要**: ユーザーの回答に応じて、性格タイプ（指揮官、仲介者など）を診断する対話型アプリケーションです。状態管理（State）を用いた条件分岐ロジックで実装しています。
* **技術**: React

## 技術スタック (Tech Stack)

* **Framework / Library**: Next.js, React
* **Languages**: JavaScript, TypeScript, YAML, HTML/CSS
* **Tools**: Figma, GitHub, Tailwind CSS
